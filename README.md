AI Crisis Intelligence Pipeline

A lightweight AI system for classifying crisis messages and generating decision strategies using advanced prompting techniques.

🚀 Features
Message classification (LLM-based)
Decision strategy generation
Token-aware processing
Structured outputs (JSON, reports)

🧠 Techniques Used
Few-Shot Prompting → Improves classification accuracy
Chain of Thought (CoT) → Step-by-step reasoning
Tree of Thought (ToT) → Multi-path decision exploration
